## Build instructions

Make sure babel, webpack and webpack-dev-server are installed globally:

	npm install -g babel webpack webpack-dev-server

Install npm dependencies:

	npm install

Build using (output is in `www`):
	
	webpack

Or run watch with:

	npm start
